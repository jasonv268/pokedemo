TP2: Pokedemo

lancer le projet:

git clone

cd pokedemo

npm install

ng serve

Fonctionnalités:

-Recherche textuelle d'un pokemon dans la liste de 20 pokemons
-Obtention de son ID
-Obtention de ses capacités
-Bouton go avec style primeng
-Calendrier primeng
